\CWHeader{Лабораторная работа \textnumero 2}\CWProblem{Необходимо спроектировать и запрограммировать на языке С++ класс-контейнер первого уровная, содержащий одну фигуру, согласно варианту задания \parКлассы должны удовлетворять следующим правилам:\begin{itemize}\item{Требования к классу фигуры аналогичны требованиям из лабораторной работы 1.}\item{Классы фигур должны иметь переопределенный оператор вывода в поток std::ostream(<<). Оператор должен распечатывать параметры фигуры.}\item{Классы фигур должны иметь переопределенный оператор ввода фигуры из потока std::istream(>>). Оператор должен вводить параметры фигуры.}\item{Классы фигур должны иметь операторы копирования (=).}\item{Классы фигур должны иметь операторы сравнения с такими же фигурами (==).}\item{Класс-контейнер должен содержать объекты фигур "по значению" (не по ссылке).}\item{Класс-контейнер должен иметь метод по добавлению фигуры в контейнер.}\item{Класс-контейнер должен иметь методы по получению фигуры из контейнера.}\item{Класс-контейнер должен иметь метод по удалению фигуры из контейнера. }\item{Класс-контейнер должен иметь перегруженный оператор по выводу контейнера в поток std::ostream(<<).}\item{Класс-контейнер должен иметь деструктор, удаляющий все элементы контейнера.}\item{Классы должны быть расположены в раздельных файлах: отдельно заголовки (.h), отдельно описание методов (.cpp).}\end{itemize}{\bfseries Фигура:} трапеция. \\{\bfseries Контейнер:} связный список.}\pagebreak
