\CWHeader{Лабораторная работа \textnumero 3}\CWProblem{Необходимо спроектировать и запрограммировать на языке С++ класс-контейнер первого уровня, содержащий все три фигуры, согласно варианту задания \parКлассы должны удовлетворять следующим правилам:\begin{itemize}\item{Требования к классу фигуры аналогичны требованиям из лабораторной работы 1.}\item{Класс-контейнер должен содержать объекты, используя std::shared ptr<...>.}\item{Класс-контейнер должен иметь метод по добалению фигуры в контейнер.}\item{Класс-контейнер должен иметь методы по получению фигуры из контейнера.}\item{Класс-контейнер должен иметь метод по удалению фигуры из контейнера.}\item{Класс-контейнер должен иметь перегруженный оператор по выводу контейнера в поток ostream.}\item{Класс-контейнер должен иметь деструктор, удаляющий все элементы контейнера.}\item{Классы должны быть расположены в раздельных файлах: отдельно заголовки (.h), отдельно описание методов (.cpp).}\end{itemize}{\bfseries Фигурs:} трапеция, ромб, пятиугольник \\{\bfseries Контейнер:} связный список.}\pagebreak
